# 📊 Exploratory Data Analysis — IPL 2025
### CodeAlpha Data Analytics Internship — Task 2

## 📌 Objective
Perform EDA on IPL 2025 batting statistics to uncover
patterns, trends and insights about player performance.

## 🛠️ Tools & Libraries
Python | Pandas | NumPy | Matplotlib | Seaborn

## 📁 Files
- `Task2_EDA.py` — Complete EDA script
- `IPL2025Batters.csv` — Raw dataset (156 players)
- `IPL2025_Cleaned.csv` — Cleaned dataset

## ▶️ How to Run
pip install pandas numpy matplotlib seaborn
python Task2_EDA.py

## ❓ Questions Answered
1. Who are the top 10 run scorers?
2. Which team scored the most runs?
3. Who has the best batting average?
4. Who has the best strike rate?
5. Correlation between batting metrics
6. Which team hit the most sixes?
7. Run distribution across all players

## 🏏 Key Findings
- Sai Sudharsan (GT) — Top scorer with 759 runs
- PBKS — Highest team total runs
- Runs vs Balls Faced — strongest correlation (0.99)
- Only 9 players scored a century in IPL 2025
